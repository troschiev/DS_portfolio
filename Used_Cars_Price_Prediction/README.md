# Определение стоимости автомобилей

Проект посвящен предсказанию стоимости БУ автомобиля по его характеристикам, указанным владельцем.

Цель:
- разработать модель, предсказывающую стоимость БУ автомобиля с минимальной метрикой RMSE

Для достижения цели необходимо выполнить задачи:
- загрузка и ознакомление с датасетом
- EDA и предобработка данных: очистка от выбросов, объединение категорий
- реализация различных моделей на основе градиентного бустинга
- обучение моделей, подбор гиперпараметров, сравнение результатов
